# 🐷 Projeto Cofrinho

## 🎯 Objetivo
O projeto **Cofrinho** foi desenvolvido como exercício prático de **programação orientada a objetos (POO)** em Python.  
Ele simula um cofrinho digital, permitindo guardar dinheiro, consultar saldo e interagir com opções de uso.

---

## 📚 Estudos aplicados

- **[Classes e Objetos](ca://s?q=Estudo_de_classes_e_objetos_em_Python)**  
  Estrutura principal do projeto, representando o cofrinho como um objeto.

- **[Método construtor __init__](ca://s?q=Estudo_do_metodo___init__em_Python)**  
  Inicializa o saldo em zero ao criar um novo cofrinho.

- **[Atributos de instância](ca://s?q=Estudo_de_atributos_de_instancia_em_Python)**  
  `self.saldo` guarda o valor acumulado de cada cofrinho.

- **[Métodos de classe](ca://s?q=Estudo_de_metodos_de_classe_em_Python)**  
  Funções como `guardar_dinheiro`, `ver_saldo` e `quebrar_cofrinho` permitem manipular o saldo.

- **[Estruturas de decisão](ca://s?q=Estudo_de_estruturas_de_decisao_em_Python)**  
  Uso de `if/else` para validar valores e controlar opções do menu.

- **[Laços de repetição](ca://s?q=Estudo_de_loops_em_Python)**  
  O `while True` permite depósitos múltiplos até o usuário decidir encerrar.

- **[Tratamento de exceções](ca://s?q=Estudo_de_tratamento_de_excecoes_em_Python)**  
  O bloco `try/except` evita erros quando o usuário digita valores inválidos.

- **[Interação com usuário](ca://s?q=Estudo_de_input_em_Python)**  
  Uso de `input()` para receber valores e escolhas diretamente do usuário.

---

## 🛠 Funcionalidades implementadas
- Depositar dinheiro no cofrinho.  
- Perguntar se deseja continuar ou encerrar.  
- Finalizar sem mostrar saldo (mensagem de sucesso).  
- Quebrar o cofrinho e revelar o saldo acumulado.
